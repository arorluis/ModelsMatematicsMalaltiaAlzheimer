# ModelsMatematicsMalaltiaAlzheimer
Alguns dels codis emprats en este TFG
prueba_maji.m i prueba_model7.m corresponen a les simulacions dels models Maji i proposat.
subsistema_numeric.m és la representació de les solucions trobades analíticament per a Q,C.
nou_escombratge_lambda3.m correspon al gràfic de la discussió del valor llindar $\bar\lambda_3$
terapia_amiloide_B.m és un exemple de com extreiem les gràfiques d'evolució temporal i efecte de les teràpies (cas anti-oligòmers)
els altres tres codis són les definicions dels models.
